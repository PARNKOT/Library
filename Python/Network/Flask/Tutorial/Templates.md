# Шаблоны
1) from flask import render_template
2) Используется шаблонизатор jinja2
3) Шаблонизатор заменяет в файле динамические компоненты на статические
    - {{ component }}
4) return render_template("file", **kwargs) // kwargs = {component: value, ... }
